# 输入逻辑测试

## 测试目标
验证输入的内容只在按 Enter 提交后才显示在网格中，并且显示正确的颜色标识

## 测试步骤

### 1. 输入过程测试
- [ ] 打开 http://127.0.0.1:5177/
- [ ] 在输入框中输入 "风调雨顺"
- [ ] 确认网格中仍然显示空行，没有显示输入的内容
- [ ] 确认输入框中显示完整的 "风调雨顺"

### 2. 提交测试
- [ ] 输入 "风调雨顺" 后按 Enter 键
- [ ] 确认输入的内容被添加到网格中
- [ ] 确认显示正确的颜色标识（青色、橙色或灰色）
- [ ] 确认自动添加新的空行
- [ ] 确认输入框被清空

### 3. 输入法测试
- [ ] 使用输入法输入 "班门弄斧"
- [ ] 确认输入过程中可以看到输入法的中间状态
- [ ] 确认选择词语后输入框显示完整的 "班门弄斧"
- [ ] 按 Enter 键提交
- [ ] 确认输入的内容被添加到网格中并显示颜色标识

### 4. 错误输入测试
- [ ] 输入少于 4 个字符（如 "风雨"）
- [ ] 按 Enter 键
- [ ] 确认显示错误消息 "请输入四个字"
- [ ] 确认网格中没有添加内容
- [ ] 确认输入框内容保持不变

### 5. 颜色标识测试
- [ ] 输入与答案有相同汉字的成语
- [ ] 确认正确位置的拼音部分显示青色
- [ ] 输入与答案有相同声母的成语
- [ ] 确认声母部分显示正确的颜色
- [ ] 输入与答案有相同韵母的成语
- [ ] 确认韵母部分显示正确的颜色
- [ ] 输入与答案有相同声调的成语
- [ ] 确认声调部分显示正确的颜色

## 预期结果
- 输入的内容只在按 Enter 提交后才显示在网格中
- 提交后显示正确的颜色标识
- 输入法可以正常使用，中间状态可见
- 错误输入时显示提示信息
