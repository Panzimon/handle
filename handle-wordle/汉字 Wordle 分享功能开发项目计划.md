# 汉字 Wordle 分享功能开发项目计划

## 一、项目背景

### 1.1 项目概述
本项目为汉字 Wordle 游戏（汉兜）添加分享功能，允许用户将游戏结果以文本或图片形式分享至社交媒体。该功能参考 `https://handle.antfu.me/` 的设计，旨在提升用户参与度和游戏传播性。

### 1.2 现有功能
- 四字成语猜测游戏
- 拼音提示系统（拼音提示 + 汉字提示）
- 速查表功能
- 游戏结果颜色标识（绿色/橙色/灰色）

### 1.3 新增功能需求
- **文本复制分享**：生成 Emoji 矩阵文本，一键复制到剪贴板
- **图片保存分享**：将游戏结果生成为精美图片，支持下载保存
- **提示状态展示**：在分享内容中明确显示用户是否使用提示及使用程度

---

## 二、项目目标

### 2.1 功能目标
| 序号 | 功能模块 | 目标描述 | 优先级 |
|-----|---------|---------|-------|
| 1 | 文本分享 | 生成包含 Emoji 矩阵的分享文本，支持一键复制 | P0 |
| 2 | 图片分享 | 生成精美分享图片，包含游戏网格和提示状态 | P0 |
| 3 | 提示状态追踪 | 记录用户使用提示的情况（无提示/拼音提示/汉字提示） | P0 |
| 4 | 状态展示 | 在分享内容中直观展示提示使用状态 | P0 |

### 2.2 用户体验目标
- 分享操作简洁，一键完成
- 提示状态展示清晰直观
- 分享内容美观，适合社交媒体传播
- 功能稳定，兼容主流浏览器

### 2.3 技术目标
- 代码结构清晰，易于维护
- 状态管理规范，避免数据混乱
- 性能优化，图片生成快速流畅

- 技术方案设计文档
- UI/UX 设计规范

---

### 阶段二：状态管理改造（第 1 阶段）
**时间预估**：1 天

**主要工作**：
1. 扩展类型定义，新增提示使用状态类型
2. 修改 useGame Hook，集成提示状态管理
3. 在提示功能调用点更新状态
4. 在游戏重置时清空状态

**关键逻辑**：
- 提示级别定义：0（无提示）、1（拼音提示）、2（汉字提示）
- 状态更新时机：用户点击提示按钮时更新级别
- 状态重置时机：新游戏开始或切换词语时重置

---

### 阶段三：文本分享功能开发（第 2 阶段）
**时间预估**：1 天

**主要工作**：
1. 创建文本分享工具模块
2. 实现分享文本生成函数
3. 实现剪贴板复制功能（现代 API + 降级方案）
4. 集成到主界面，添加分享按钮

**分享文本格式**：
```
汉兜 · 第 XXX 期
[提示状态标识]

[Emoji 矩阵]

[游戏结果]
```

**提示状态标识**：
- 无提示：`🏆 无提示辅助`
- 拼音提示：`🔤 拼音提示`
- 汉字提示：`💡 汉字提示`

---

### 阶段四：图片分享功能开发（第 3 阶段）
**时间预估**：2 天

**主要工作**：
1. 引入 html2canvas 依赖库
2. 创建图片生成工具模块
3. 设计分享图片的视觉样式
4. 实现图片生成和下载功能
5. 集成到主界面

**图片内容结构**：
- 顶部：游戏标题（汉兜 · 第 X 期）
- 上部：提示状态徽章（带颜色和图标）
- 中部：游戏结果网格（4xN 矩阵）
- 底部：游戏结果文字

**视觉设计规范**：
- 背景：渐变色（紫色系）
- 提示徽章：圆角矩形，根据级别显示不同颜色
  - 无提示：翠绿色
  - 拼音提示：琥珀色
  - 汉字提示：红色
- 网格单元格：圆角，颜色标识（绿/橙/灰）

---

### 阶段五：UI 集成与优化（第 4 阶段）
**时间预估**：1 天

**主要工作**：
1. 在游戏结束界面添加分享按钮组
2. 实现按钮交互和反馈（Toast 提示）
3. 响应式适配，确保移动端体验
4. 添加加载状态和错误处理

**按钮布局**：
```
[再玩一次] [复制结果] [保存图片] [换一个词]
```

**交互反馈**：
- 复制成功：Toast 提示"已复制到剪贴板"
- 保存成功：Toast 提示"图片已保存"
- 操作失败：Toast 提示错误信息

---

### 阶段六：测试与优化（第 5 阶段）
**时间预估**：2 天

**主要工作**：
1. 功能测试：验证所有分享场景
2. 兼容性测试：多浏览器验证
3. 性能测试：图片生成速度优化
4. 边界测试：异常情况和错误处理

**测试场景**：
| 场景 | 预期结果 |
|-----|---------|
| 无提示通关 | 显示"🏆 无提示辅助" |
| 使用拼音提示后通关 | 显示"🔤 拼音提示" |
| 使用汉字提示后通关 | 显示"💡 汉字提示" |
| 未通关（失败） | 正常显示提示状态 |
| 新游戏开始 | 重置为无提示状态 |
| 切换词语 | 重置为无提示状态 |
