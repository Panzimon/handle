# 手动测试步骤

## 测试目标
验证汉字 Wordle 游戏的核心功能是否正常工作

## 测试步骤

### 1. 页面加载测试
- [ ] 打开 http://127.0.0.1:5175/
- [ ] 确认页面标题显示 "汉字 Wordle"
- [ ] 确认游戏说明文字显示正确
- [ ] 确认 10 行 4 列的游戏网格显示正确

### 2. 输入功能测试
- [ ] 直接在页面上输入汉字 "班门弄斧"
- [ ] 确认输入框中显示正确的汉字
- [ ] 确认输入长度限制为 4 个字符
- [ ] 确认只能输入汉字，其他字符会被过滤

### 3. 提交功能测试
- [ ] 输入 "班门弄斧" 后按 Enter 键
- [ ] 确认显示胜利消息 "恭喜你，猜对了！"
- [ ] 确认所有格子变为青色

### 4. 错误输入测试
- [ ] 输入少于 4 个字符（如 "班门"）
- [ ] 按 Enter 键
- [ ] 确认显示错误消息 "请输入四个字"
- [ ] 确认当前行有摇晃动画

### 5. 重新开始功能测试
- [ ] 完成一局游戏后，点击 "再玩一次" 按钮
- [ ] 确认游戏重置，输入框清空
- [ ] 确认新的答案生成

### 6. 颜色反馈测试
- [ ] 输入包含正确位置汉字的成语
- [ ] 确认正确位置的格子显示青色
- [ ] 输入包含存在但位置错误汉字的成语
- [ ] 确认存在但位置错误的格子显示橙色
- [ ] 输入包含不存在汉字的成语
- [ ] 确认不存在的格子显示灰色

### 7. 拼音反馈测试
- [ ] 输入与答案有相同声母的成语
- [ ] 确认声母部分显示正确的颜色
- [ ] 输入与答案有相同韵母的成语
- [ ] 确认韵母部分显示正确的颜色
- [ ] 输入与答案有相同声调的成语
- [ ] 确认声调部分显示正确的颜色

## 预期结果
所有测试步骤都应通过，游戏功能与原网站一致。
