# 删除功能修复验证测试

## 测试目标
验证删除功能是否正常工作：
1. 选中文本删除功能
2. 光标定位删除功能
3. 不同字符长度和光标位置下的删除操作

## 测试步骤

### 1. 选中文本删除功能测试
- [ ] 打开 http://127.0.0.1:5173/
- [ ] 在输入框中输入 "风调雨顺"
- [ ] 用鼠标选中 "调雨" 两个字符
- [ ] 按 Backspace 键删除选中的文本
- [ ] 确认输入框显示 "风顺"
- [ ] 用鼠标选中 "风顺" 两个字符
- [ ] 按 Backspace 键删除选中的文本
- [ ] 确认输入框为空

### 2. 光标定位删除功能测试
- [ ] 在输入框中输入 "风调雨顺"
- [ ] 将光标移动到 "雨" 字符后（即第三个字符后）
- [ ] 按 Backspace 键
- [ ] 确认输入框显示 "风调顺"
- [ ] 将光标移动到 "调" 字符后（即第二个字符后）
- [ ] 按 Backspace 键
- [ ] 确认输入框显示 "风顺"
- [ ] 将光标移动到 "风" 字符后（即第一个字符后）
- [ ] 按 Backspace 键
- [ ] 确认输入框显示 "顺"

### 3. 不同字符长度测试
- [ ] 在输入框中输入 "风"
- [ ] 按 Backspace 键
- [ ] 确认输入框为空
- [ ] 在输入框中输入 "风雨"
- [ ] 按 Backspace 键
- [ ] 确认输入框显示 "风"
- [ ] 按 Backspace 键
- [ ] 确认输入框为空
- [ ] 在输入框中输入 "风调雨"
- [ ] 按 Backspace 键
- [ ] 确认输入框显示 "风调"

### 4. 边界情况测试
- [ ] 在输入框为空时，按 Backspace 键
- [ ] 确认输入框保持为空
- [ ] 在输入框中输入 "风调雨顺" 后，将光标移动到最前面
- [ ] 按 Backspace 键
- [ ] 确认输入框显示 "调雨顺"

## 预期结果
- 选中文本后执行删除操作时，能够准确移除所有选中的字符
- 光标定位状态下执行删除操作时，能够精准删除光标位置前的单个字符
- 边界情况下的删除操作正常
- 所有删除操作符合标准文本编辑操作逻辑
