# 文字存在但位置不正确时的样式验证测试

## 测试目标
验证当文字存在于答案中但位置不正确时的样式显示是否符合要求：
1. 文字框的背景颜色是否为橙色
2. 文字颜色是否为白色
3. 拼音颜色是否为白色
4. 其他匹配情况的样式是否保持不变

## 测试步骤

### 1. 完全正确情况测试
- [ ] 打开 http://127.0.0.1:5173/
- [ ] 输入与答案完全匹配的成语
- [ ] 确认文字框底色为绿色 (#6aaa64)
- [ ] 确认文字为白色
- [ ] 确认拼音为白色
- [ ] 确认声调为白色

### 2. 部分匹配情况测试
- [ ] 输入部分匹配的成语（文字存在于答案中但位置不对）
- [ ] 确认文字框底色为橙色 (#c9b458)
- [ ] 确认文字为白色
- [ ] 确认拼音为白色
- [ ] 确认声调为白色

### 3. 完全不匹配情况测试
- [ ] 输入完全不匹配的成语
- [ ] 确认文字框底色为灰色 (#787c7e)
- [ ] 确认文字为白色
- [ ] 确认拼音颜色保持现有设置（白色）
- [ ] 确认声调颜色保持现有设置（白色）

### 4. 混合情况测试
- [ ] 输入混合匹配的成语（部分正确，部分存在但位置不对，部分不存在）
- [ ] 确认文字框底色为灰色
- [ ] 确认文字为白色
- [ ] 确认正确的拼音为白色
- [ ] 确认存在但位置不对的拼音为白色
- [ ] 确认不存在的拼音为白色

## 预期结果
- 完全正确时：文字框绿色，文字白色，拼音白色
- 部分匹配时：文字框橙色，文字白色，拼音白色
- 完全不匹配时：文字框灰色，文字白色，拼音白色
- 混合情况时：文字框灰色，文字白色，拼音白色
