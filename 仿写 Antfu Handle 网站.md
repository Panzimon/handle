### 功能特点

1. **游戏核心**
   - 每日生成一个四字成语作为答案
   - 玩家有 10 次猜测机会
   - 支持键盘输入和虚拟键盘输入
2. **颜色反馈系统**
   - 🟩 **青色 (correct)**: 汉字/声母/韵母/声调 完全正确
   - 🟨 **橙色 (present)**: 元素存在于答案中但位置不对
   - ⬜ **灰色 (absent)**: 元素不存在于答案中
3. **详细反馈**
   - 每个格子独立显示汉字、声母、韵母、声调
   - 每个组件都有独立的颜色标识
4. **词库**
   - 包含 600+ 个常用四字成语
   - 支持每日更新和随机选择
5. **交互体验**
   - 行摇晃动画（输入无效时）
   - 响应式设计，支持移动端
   - 游戏结束后可重新开始

### 技术栈

- **Vite** + **React** + **TypeScript**
- **pinyin** 库处理汉字拼音
- 纯 CSS 样式，无 UI 框架依赖
